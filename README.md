# MCServerManager

Minecraft Java Edition のサーバー管理を、誰でも直感的に・全自動で行えるようにするWindows専用デスクトップアプリケーションです。

「ポート開放が難しくて友達を呼べない」「黒い画面（コマンドプロンプト）の操作がよくわからない」といった悩みを解決し、ボタン一つでマルチプレイ環境を構築します。

## ✨ 主な特徴

- **🚀 ワンボタンで全自動起動**
  起動ボタンを押すだけで、「ファイアウォールの開放」「UPnPによるルーターのポート開放（ポートマッピング）」「サーバーの起動」をすべて自動で行います。
- **👥 リアルタイム監視**
  サーバーが起動しているか、現在何人のプレイヤーが遊んでいるかを10秒ごとに自動更新してリストに表示します。
- **🌐 IPアドレスのワンタッチコピー**
  マルチプレイで友達に教える必要がある「グローバルIPアドレス」を、ボタン一つで取得してクリップボードにコピーできます。
- **🧠 スマートな起動処理**
  面倒なUAC（管理者権限）プロンプトは、初回設定時のみ表示。2回目以降は自動でスキップする賢い設計です。

## 📦 ダウンロードと導入

本アプリは、インストール不要ですぐに使える「単一の実行ファイル（Self-contained）」として配布されています。

1. [Releasesページ](../../releases) から、最新バージョンの `MCServerManager.exe` をダウンロードします。
2. ダウンロードした `.exe` ファイルを好きなフォルダ（デスクトップなど）に配置します。
3. ダブルクリックで起動すれば、すぐに使い始められます！（.NETなどの事前インストールは不要です）

---

## 🎮 使い方

1. アプリを起動し、左上の **「➕ サーバー追加」** ボタンをクリックします。
2. お手持ちのMinecraftサーバーが保存されている **「フォルダ」** と、起動用の **「.batファイル」**（例: `start.bat`）を選択して登録します。
3. リストに追加されたサーバー行の **「起動」** ボタンを押します。
4. 初回のみファイアウォール設定のための確認（はい/いいえ）が出ます。「はい」を押すと、ポート開放とサーバー起動が自動で完了します！
5. **「IPコピー」** ボタンを押して自身のIPをコピーし、一緒に遊ぶ友達に教えてあげましょう。

> **※UPnPに関する注意**
> お使いのルーターがUPnP機能に対応していない、または無効化されている場合は、自動でのポート開放が失敗することがあります。その場合はルーターの設定画面から手動でポート（デフォルト: 25565）を開放してください。

## 🛠️ 開発者向け情報

本プロジェクトはC# / WPF (.NET 8) で開発されています。ご自身でビルドやカスタマイズを行いたい方向けの情報です。

### 使用技術
- **フレームワーク**: .NET 8.0 WPF (MVVMパターン)
- **ネットワーク制御**: [SharpOpenNat](https://github.com/lviranyi/SharpOpenNat) (UPnP通信用)
- **マイクラ通信**: [MineStat](https://github.com/fragmer/minestat) (Server List Pingによるプレイヤー数取得)

### ビルド方法
Visual Studio 2022 で `.sln` を開いてビルドするか、以下のコマンドで単一ファイルとして出力可能です。

```powershell
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```

### プロジェクト構成
- `Models/`: `ServerInfo.cs` (サーバー状態のデータモデル)
- `ViewModels/`: `MainViewModel.cs` (リスト管理、ポーリングロジック)
- `Services/`: `ServerManagerService.cs` (プロセス起動、ファイアウォール制御、UPnPマッピング)
